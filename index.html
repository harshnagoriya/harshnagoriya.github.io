Projects

  • FaceMask Detection System

    • Technology: Machine Learning, Deep Learning, OpenCV, Flask, Cloud Computing, IoT, Python

    In the wake of current corona virus pandemic situation, it is necessary for all to wear a mask to thwart the event of cross-contamination, So, I worked on a project of Mask Detection System based on Machine Learning and OpenCV, in which I created dataset of people wearing mask and without mask, the recognizes whether the person has worn mask or not, all task is mechanized, no human interaction is required. I trained 101 layers deep convolutional neural network, Resnet-101 on the data I gathered and integrated it with Flask for building web application.

  • Helmet Detection System

    • Technology: Machine Learning, Deep Learning, OpenCV, YOLO, SSD, Python

    The higher death rate in motorbike accidents is credited to carelessness in wearing a helmet by bike riders. Identification of helmetless riders continuously is a necessary task to forestall the event of such accidents, So, I worked on a project of Helmet Detection System based on Deep Learning and OpenCV which recognizes whether the rider has worn mask or not, if not, his numberplate is captured and data is stored to database. All task is automated, I used SSD for finding bikers among moving objects, after that, I used CNN to find out whether biker has worn helmet or not, if not, His numberplate is captured via YOLO & Tesseract.

  • Student Attendance System using face Recognition

    • Technology: Machine Learning, Deep Learning, OpenCV, RaspberryPI, Python

    Taking attendance is a real-world task, which needs a creative solution to reduce time, efforts and resources. In this project, a raspberry PI based system detects and recognize human faces in a quick and accurate way via images or videos that are being captured through a Camera. It detects the faces within the image and compares it with the listed faces in the database. On recognition of a registered face on the captured image assortments, the attendance of that student is marked present otherwise absent. Eigen matrix concept (Eigen Face) is used in this project.

  • Handwritten Gujarati Characters Recognition

Then I made prediction among 34 classes of characters.
